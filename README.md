# How To Use
Environment: ruby 2.5.3, rails 5.2.2

## Download git repo
- git clone https://github.com/WeiChengChiu/membership.git

## Build and Run Rails project
- cd membership
- bundle
- rake db:migrate
- rake db:seed (before run rspec)
- rails server
